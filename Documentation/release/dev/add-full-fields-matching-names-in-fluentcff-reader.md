# Add full fields matching names for **FLUENTCFFReader**

When reading a FluentCFF file, the arrays fields names can now be replaced by a more meaningful name that correspond to what Fluent displays
if the `RenameArrays` option is enabled. The details for these renamings can be found [here](https://ansyshelp.ansys.com/public/account/secured?returnurl=/Views/Secured/corp/v242/en/flu_udf/flu_udf_DataAccessMacros.html) and [here](https://developer.ansys.com/docs/common-fluids-format-/_data_models_overview.html)
