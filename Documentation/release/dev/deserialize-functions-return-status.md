# Deserialization functiosn return status

When deserializing data, the autogenerated and manually coded `Deserialize_vtkClassName` functions will return a `bool` status indicating the success (`true`) or failure (`false`) of the operation. This status can help developers handle errors and ensure that the data has been correctly processed.

## New Signatures

Earlier, these methods returned `void`. The updated signatures for the deserialization functions are as follows:

```c++
1. bool DeserializeJSON(const vtkTypeUInt32& identifier, vtkSmartPointer<vtkObjectBase>& objectBase);
2. bool UpdateObjectFromState(const std::string& state);
3. bool UpdateObjectFromState(const nlohmann::json& state);
```
