## Deprecate `VTK_LEGACY_REMOVE`

The following CMake variable have been deprecated:
`VTK_LEGACY_REMOVE`
`VTK_LEGACY_SILENT`

The `vtkLegacy.h` header have been deprecated and the related defines are now fixed and will be removed in the future:
`VTK_LEGACY_REMOVE`
`VTK_LEGACY_SILENT`
`VTK_LEGACY_BODY`
`VTK_LEGACY_REPLACED_BODY`
